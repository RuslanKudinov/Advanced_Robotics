# Домашнее задание 2: Оценка пройденного пути с помощью фильтра Калмана

## 📋 Описание проекта
В этой работе реализован линейный фильтр Калмана для объединения данных GPS и акселерометра с целью точной оценки пройденного пути. Использованы данные из датасета KITTI (sequence 2011_09_26_drive_0001_sync).

## 🗂 Структура репозитория
- `Kalman_Filter_KITTI.ipynb` — Jupyter ноутбук с полной реализацией
- `kf_results.csv` — выходные данные (дистанция по GPS, оценка фильтра Калмана, скорость)
- `README.md` — описание проекта

## 📊 Результаты

### Ключевые метрики
| Параметр | Значение |
|----------|----------|
| Общая дистанция по GPS | 107.11 м |
| Общая дистанция по KF | 107.08 м |
| Разница | 0.04 м (4 см!) |
| Среднеквадратичная ошибка (RMSE) | 0.04 м |
| Шум измерения GPS (std_meas) | 0.0655 м |
| Шум акселерометра (std_acc) | 0.1289 м/с² |
| Количество измерений | 108 |

### 📈 Визуализация
Ноутбук содержит несколько графиков:
1. **Пройденный путь** — сравнение сырых GPS измерений и отфильтрованной траектории
2. **Начальный участок** — зум на первые измерения для демонстрации работы фильтра
3. **Ошибка оценки** — разница между GPS и KF во времени
4. **Скорость** — оценка скорости фильтром Калмана

## 🔧 Реализация фильтра Калмана

### Модель пространства состояний
- **Вектор состояния**: x = [position, velocity]ᵀ
- **Управление**: u = ускорение из акселерометра (ось X)


### Оценка шумов
- **std_meas** (шум GPS): рассчитан как стандартное отклонение приращений GPS на начальном участке (предположительно, объект стоял)
- **std_acc** (шум акселерометра): стандартное отклонение показаний акселерометра в покое

## 🚀 Как запустить
1. Открыть `Kalman_Filter_KITTI.ipynb` в Google Colab
2. Запустить все ячейки последовательно
3. Данные скачаются автоматически с Google Drive по предоставленной ссылке

## 💡 Выводы
Фильтр Калмана успешно объединил зашумленные GPS измерения и данные акселерометра. Результирующая траектория более гладкая, при этом итоговая дистанция практически совпадает с GPS (ошибка 4 см на 107 метрах). Это демонстрирует эффективность фильтрации для задач оценки положения.
